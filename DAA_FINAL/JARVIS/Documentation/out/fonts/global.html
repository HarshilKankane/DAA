Brief Explanation of geom.js

This file defines a 2D vector class called Vec along with utility functions for vector operations. It includes methods for vector addition, subtraction, dot product, checking if three points make a counter-clockwise turn, checking if vectors are parallel, and calculating the square of the length of a vector. Additionally, it exports functions isCcw and isCcwA which determine if three points make a counter-clockwise turn.

Brief Explanation of plotter.js

This file provides a function for plotting points using the Plotly library. It takes an array of points, a mode for plotting (e.g., 'markers', 'lines', 'lines+markers'), and styling options as input. It supports plotting points represented either as instances of the Vec class or as arrays. The function generates a plot with x and y axes ranging from -1 to 101 and exports the function plot.
